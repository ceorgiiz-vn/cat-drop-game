# Cat Drop 2 — Соревновательная версия (изолированная копия)

Это **отдельная, независимая копия** игры для экспериментов с соревновательными фичами.

- Сама игра — в `www/` (полная копия основной игры на момент создания).
- Оригинал лежит рядом, в `../www/`, и **не изменяется**. Что бы здесь ни ломалось — оригинал в безопасности.
- **Откат:** просто удалить папку `competitive-edition` целиком. Основная игра не пострадает.
- У копии свой PWA-кэш (`catdrop-competitive-v1`) и своё имя («Cat Drop 2 — Competitive»), чтобы не пересекаться с оригиналом.

План работ: см. `ROADMAP_соревновательная.md`.

Когда фича здесь проверена и нравится — переносим её в основную игру отдельным шагом.
