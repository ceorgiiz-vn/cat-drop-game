# Cat Drop 2 — Соревновательная версия · Дорожная карта

Это изолированная копия игры (`competitive-edition/`). Оригинал (`../www/`) не трогаем — откат = просто удалить эту папку.

Цель: добавить «соревновательный крючок», чтобы игрок не просто ронял котов, а возвращался соревноваться — с собой, с другими, каждый день.

Все фичи разбиты по волнам: сверху — то, что работает **без сервера** (внедряем здесь и сразу тестируем), снизу — то, что требует бэкенда/Google Play Console.

---

## Волна 1 — без сервера, играется офлайн (делаем здесь)

### 1.1 Комбо-множитель за серию слияний ⭐ рекомендую начать
Каждое слияние в течение ~2 сек после предыдущего повышает множитель (x2, x3, …). Очки идут ×множитель. На экране — счётчик комбо и «COMBO x4!».
- Зачем: даёт скилловый потолок. Два игрока с одинаковой доской наберут разное число очков → есть за что соревноваться.
- Сложность: низкая. Только `game.js` (логика слияния + HUD-индикатор). Данные не меняем.

### 1.2 Ежедневный вызов (одинаковая доска для всех) ⭐ сильнейший крючок
Последовательность котов задаётся детерминированным «сидом» от даты (сегодня у всех одинаковая). Отдельная кнопка «Челлендж дня», свой счёт за день.
- Зачем: честное сравнение — «все играли одну доску». Идеально ложится на ячейку TODAY и на шэринг результата.
- Сложность: средняя. Нужен seeded-random вместо `Math.random` в `game_modes.js`, отдельный режим и экран результата.

### 1.3 Ачивки / бейджи
Набор целей: дойти до 11 ур., сделать 50 слияний за игру, комбо x5, набрать 5000 и т.д. Показ в профиле.
- Зачем: цели удержания, «ещё один заход».
- Сложность: низкая-средняя. Хранение в `state.js` + экран в профиле.

### 1.4 Гонка с призраком собственного рекорда
Тонкая линия/маркер «твой лучший результат к этому моменту» — обгоняешь себя.
- Зачем: одиночная соревновательность без соперников.
- Сложность: низкая.

### 1.5 Челлендж другу по ссылке (сид)
Кнопка «Бросить вызов»: генерит ссылку с сидом сегодняшнего дня («побей мои 4200»). Друг открывает — та же доска.
- Зачем: вирусность без сервера (сид зашит в ссылку).
- Сложность: низкая (поверх 1.2).

## Волна 2 — требует Google Play Console / бэкенда (позже, вне песочницы)

### 2.1 Онлайн-лидерборд Google Play Games
Каркас уже в проекте (`play_games.js`), нужно завести сервис в Play Console, вставить настоящий leaderboard ID + `google-services.json`. Недельный лидерборд с обнулением по понедельникам = постоянная гонка.

### 2.2 Лиги / дивизионы (Бронза → Алмаз)
Игроки распределяются по лигам, недельный счёт двигает вверх/вниз. Самый «липкий» механизм удержания. Нужен серверный подсчёт или GPGS + логика.

### 2.3 Еженедельные турниры с наградами
Победа в недельном лидерборде → косметика (скины/темы, которые уже есть в экономике). Связывает соревнование с магазином.

## Рекомендуемый порядок внедрения
1. **Комбо-множитель** (1.1) — быстро, ощущается сразу, углубляет очки.
2. **Ежедневный вызов** (1.2) — главный крючок, детерминированная доска.
3. **Челлендж другу** (1.5) — вирусность поверх 1.2.
4. **Ачивки** (1.3) + **призрак** (1.4) — удержание.
5. Волна 2 (GPGS/лиги) — когда будете готовы заводить Play Console и бэкенд.

Каждый пункт внедряем **в этой папке**, тестируем как PWA, и только проверенное потом переносим в основную игру.
