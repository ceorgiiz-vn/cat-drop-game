## 1. Что за игра
**Cat Drop** — головоломка в духа Suika/2048: игрок бросает котов в стакан, два одинаковых
кота при касании **сливаются** в кота следующего уровня (1→11). Цель — набрать очки и дойти
до Кота-Космонавта (11 уровень). Если коты переполняют стакан выше красной линии дольше 3 сек — Game Over.
Есть магазин (темы/скины/звуки за рыбо-монеты), бустеры (SHAKE, POP), таблица лидеров, облачный сейв.

## 2. Стек
- **Движок физики:** Matter.js (`www/js/matter.min.js`).
- **Рендер:** HTML5 Canvas (fixed-координаты 720×1280), интерфейс — HTML/CSS поверх.
- **Обёртка:** Capacitor под Android (`android/`) + PWA (Service Worker, офлайн).
- **Сервисы:** Google Play Games (вход, лидеры, облачный сейв), Sentry (диагностика).

Если задача **требует** правки замороженной зоны — сначала предупредить владельца и объяснить, зачем.

## 6. Как связаны блоки (поток данных)
```
Ввод (тап) → Input listeners → спавн кота → Main Game Loop → Matter.js (физика)
   → коллизия одинаковых → слияние (Physics Init) → +очки (GameState) → HUD (DOM Sync)
   → сохранение (state.js: локально + cloud_save.js в облако)
Диагностика (diagnostics*.js) читает состояние через window.__DEBUG_GAME__ — НАБЛЮДАЕТ, не вмешивается.
```
