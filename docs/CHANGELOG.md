## 2026-07-18 · versionCode 9 / v2.6 (готовится к сборке)

- **Раскладка на высоких телефонах: экран заполняется целиком.** Раньше игра 9:16 вписывалась по
  ширине и оставляла пустые полосы сверху/снизу (~20% экрана), HUD тесно сидел на стакане. Теперь на
  экранах выше 9:16 обёртка во весь экран, стакан 9:16 прижат к НИЗУ, верхняя панель — к потолку
  (занимает верхнюю зону). Планшеты/широкие экраны не тронуты. Ввод точный (маппинг по getBoundingClientRect).
  Блок: раскладка. Файлы: `www/style.css` (media max-aspect + блок в конце), `www/index.html` (style v50),
  `www/sw.js` (кэш v147).

- **Фикс общего рейтинга (Google Play Games).** Вход в Play Games открывался, но не завершался →
  рейтинг оставался локальным (per-device). Причина: плагин требует вызвать `initialize()` ПЕРЕД
  входом/лидерами/сейвом, а его не вызывали. Добавлен `initialize()` перед silent-входом.
  Блок: Play Games. Файлы: `www/js/play_games.js`, `www/index.html` (v5), `www/sw.js` (кэш v146).
  ⚠️ Зависит также от публикации таблицы лидеров в Play Console.

- **Очистка проекта + управление.** Папка 1.2 ГБ → 345 МБ (удалены старые сборки, кэш, `_apk_old`, `tmp`,
  `competitive-edition` [идеи → `docs/competitive-edition/`]). `www/screenshots` вынесены в `store-listing/`
  (APK −18 МБ). Создан `AGENTS.md` (единый управляющий файл всех ИИ) + система версий; `npm run clean`.
  Заведены `docs/ARCHITECTURE.md` и этот журнал.
  Файлы: корень, `AGENTS.md`, `CLAUDE.md`, `package.json`, `scripts/clean.mjs`, `docs/`.

- **Фикс диагностики (ложные зависания).** По данным Sentry с Z Fold 4: «Long frame 5259ms» ×195 оказались
  фоновыми паузами (сворачивание приложения), а не рывками. Теперь кадр возврата из фона игнорируется
  (+ потолок 4000 мс). Убирает мусорные срабатывания.
  Файлы/блоки: `www/js/diagnostics.js` (запись кадров, наблюдатель долгих задач), `www/sw.js` (кэш v145).

## 2026-07-17 · versionCode 6→7 / v2.3→2.6 (диагностика)

- **Сбор статистики (чёрный ящик + Sentry).** Локальный Sentry SDK + `diagnostics.js` (паспорт устройства,
  FPS, детектор зависаний, скрытый счётчик по тройному тапу) + `diagnostics_watchdogs.js` (сторожа физики:
  NaN, вылет за поле, сверхскорость, крутёж, утечка тел, застрявшее слияние). Читают состояние через
  `window.__DEBUG_GAME__` — game.js не менялся.
  Файлы: `www/js/sentry.min.js`, `www/js/diagnostics.js`, `www/js/diagnostics_watchdogs.js`,
  `www/index.html`, `www/sw.js`, `www/privacy.html`.

- **Облачный сейв (Play Games Snapshots).** `cloud_save.js` как безопасная надстройка: локальный сейв остаётся
  главным, облако добавляется сверху (guarded). Хук в `state.js.save()` и `play_games.js` (после входа).
  Включены Saved Games в Play Console.
  Файлы: `www/js/cloud_save.js`, `www/js/state.js` (1 строка), `www/js/play_games.js`.
